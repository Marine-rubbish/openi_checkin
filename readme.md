# Openi 每日运行调试环境脚本

## 0. 准备工作

- 一个启智社区的账号
- 至少一个可用的调试环境

## 1. 环境配置

1. fork本仓库到你自己的账号上。

2. 打开 [云脑任务界面](https://openi.pcl.ac.cn/cloudbrains)。按 `F12` 或 `Fn+F12` 打开浏览器开发者工具。

3. 从 `网络` 里面找到 筛选出 `Fetch/XHR` 的请求，点击其中一个请求，再找到 `Cookie` 属性。把 `Cookie` 属性复制下来。
